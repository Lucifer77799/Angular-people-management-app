# People Management Frontend

Angular 8 application for managing people data.

## Setup Instructions

1. Install dependencies:
```bash
   npm install
```

2. Start development server:
```bash
   ng serve
```

3. Open browser: http://localhost:4200

## Features

- View all persons in a table
- Create new person with form validation
- Edit existing person
- Delete person with confirmation
- Responsive design

## Requirements

- Node.js 14+
- Angular CLI 8
- Backend API running on http://localhost:3000
